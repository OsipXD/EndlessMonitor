AllOnline 
=======
Это дополнение покажет полный онлайн всех ваших серверов
Версия Мониторинга должна быть не ниже v2.0 (не бета)

Установка
---------
1. Разархивировать в папку extensions
2. В файле `serverlist.php` добавьте такой список серверов 
   присутствующих в вашем конфиге, которые вы хотите 
   учитывать в общем мониторинге:
    $servers = array(
        'название_сервера',
        'название_сервера1'
    );
3. В настройку каждого сервера добавьте его максимальный
   онлайн (чтобы учитывать его когда сервер не доступен)
   Например:
    $maxonline['название_сервера'] = 100;
4. Теперь встраиваем в нужное место фрейм
    <iframe src="http://ваш_сайт.ru/путь_к_мониторингу/extensions/AllOnline/allonline.php" frameborder="0" scrolling="0" height="по_вашему_вкусу" width="100%">
    </iframe>
